# Gen and Agentic AI Curriculum

Welcome to the ultimate learning repository for Generative AI and Agentic Orchestration Systems! This curriculum is organized into two distinct, progressive sections:

1. **`gen_ai/`**: Focuses on core Generative AI foundations, including APIs, prompt engineering, RAG, memory, fine-tuning, evaluations, and production deployment.
2. **`agentic_ai/`**: Focuses on advanced agentic orchestration, including state graphs, persistence, human-in-the-loop, Corrective RAG (CRAG), Plan-and-Execute, and Multi-Agent supervisor patterns.
